## 学習メモ
- Claude Code を使えばマネタイズのチャンスがあることが分かった
- エージェントチームの設定方法は、Claude のサイトの設定をコピーしてプロンプトで指示するだけ
- Rules を使えば CLAUDE.md の内容を分割して、コンテキストを必要なときに動的に使えるため、コンテキストの圧縮につなげられる
- Skills はカスタムスラッシュコマンドが統合されているが、カスタムスラッシュコマンド（.commands ディレクトリ内の MD ファイルで指示する定型文）も引き続き使える。両方ある場合は Skills の設定が優先される
- Codex はデスクトップアプリを指しており、Cline との比較は以下の通り：
  - Cline：プロジェクトを精査して全体設計から実装まで一貫して深く行える印象
  - Codex：単発の実装や新機能の追加、バグフィックスといったピンポイントな作業で強みを発揮しそう
  - コスト面：Codex は約 3,000 円のプランで Cline の約 30,000 円プラン相当のトークン量があり、コストパフォーマンスは Codex の方が優れている
