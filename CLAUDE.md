# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

日々の学習記録（日報）を Markdown で管理するリポジトリ。

## 構造

```
YYYY/MM/DD.md   # 各日の日報ファイル（例: 2026/03/31.md）
```

## 日報フォーマット

最近のエントリは以下の形式を使用：

```markdown
## 学習内容
- トピック

## 学習時間
- Today: Xh
- Total: X,XXX.Xh

## 学習メモ
自由記述
```

※ 過去のエントリ（2023〜2024年頃）はセクション名や構成が異なる場合がある。

## コミット規約

```
YYYYMMDD の日報作成
```

例: `20260331の日報作成`

## 運用

- 1日1ファイル、日付をファイル名にする
- 年/月でディレクトリを分ける
- 言語は日本語
